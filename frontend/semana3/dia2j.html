Arrays

Los arrays son arreglos de objetos que pueden tener diferentes tipos de valores, ya sea, boleanos, numeros, caracteres, etc.

let array=["Juan", 12, true, "Maria"];
este arreglo puede cambiar siempre en cuando sepamos el indice y para saber el indice es necesario contar desde el 0.
Es decir para mostrar el elemento "Juan" debemos poner el 0 como indice.
array[0]="Juan";
Asi como podemos mostrar el elemento mediante su indice tambien podemmos modificarlo.
array[0] = "Erick"; array[1] = 14;
Esto mostrara:
let array=["Erick", 14, true, "Maria"];

Arrays parte 2

Si queremos saber cuantos elementos posee nuestro array podemos usar la propiedad length.
ejemplo:
var ourArray = [1,2,3];
console.log("La cantidad de elementos dentro del array es de" , ourArray.length);
esto devolvera 3.
y para poder cambiar algun elemento solo es cuestion de restar -1 a esta propiedad.
