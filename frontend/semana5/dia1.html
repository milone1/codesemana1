EL DOM:

1.Crear el Elemento:

var element = document.createElement("tag-name de la etiqueta.");

2. Crear el textNode(contenido);

var contenido = document.createTextNode("label-content");

3. Agregar el contenido al elemento(appendChild=agregarHijo):

element.appendChild(contenido);

4. OPCIONAL, se puede agregar atributos al elemento.

element.setAtributte("aling","center");

5. agregar elementos al contenedor:

document.getElementById("conteiner").appendChild(elemento);

var padre = document.getElementById("padre");

var padre = document.getElementsByTagName("li"[0]).parentNode;
