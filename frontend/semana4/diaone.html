Funciones.

Es un bloque de construccion fundamental. Es similar a un procedimiento, un conjunto de instrucciones que realiza una tarea o calcula un valor.

Una funcion se define: El nombre de la función.
Una lista de parámetros de la función, entre paréntesis y separados por comas.
Las declaraciones de JavaScript que definen la función, encerradas entre llaves,

ejemplo:

function name (parametro){
sentencia;
}

Las funciones tambien pueden ser anonimas, es decir puede o no tener un nombre.

FUNCIONES EN VARIABLES.

Las funciones tambien pueden ser almacenadas en variables para poder usarlas como cualquier otra variable dentro de nuestro js.

const name_variable = function name (parametro){
sentencia;
}

Arrow function

No tiene sus propios enlaces a this o super y no se debe usar como métodos.
No apta para los métodos call, apply y bind, que generalmente se basan en establecer un ámbito o alcance
No se puede utilizar como constructor.
No se puede utilizar yield dentro de su cuerpo.
No tiene argumentos o palabras clave new.target.

function flecha de una sola linea:

(parámetros) => sentencia;

funcion flecha de varias lineas:

(parámetros) =>{
sentencia;
}
