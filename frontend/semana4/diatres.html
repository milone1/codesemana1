WHILE

Estructura repetitiva o bucle. significa mientras, ya que, mientras se cumpla cierta condicion este permanecera activo.

while (condicion){
sentencia
}

FOR

La estructura repetitiva for es la que podemos controlar con mayor facilidad, ya que, entre sus condiciones podemos decir como empieza, donde para, y que hace con cada repetiticion.

for (inicia; para; hace){
sentencia
}

FOR IN

La estructura for in itera sobre todas las propiedades de un objeto.

var obj = {a: 1, b: 2, c: 3};
for (const prop in obj) {
console.log(`obj.${prop} = ${obj[prop]}`);
}
// Produce:
// "obj.a = 1"
// "obj.b = 2"
// "obj.c = 3"

FOR OF

La sentencia for of ejecuta un blqoue de codigo para cada elemento de un objeto.

let iterable = "boo";
for (let value of iterable) {
console.log(value);
}
// "b"
// "o"
// "o"
